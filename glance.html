Jobsplane is a job aggregator which helps people across India find the right jobs and internships for them. We worked with the Jobsplane team on an updated brand identity, a redesign of their website, and on the design for their new mobile app.

Identity

The website and branding for Jobsplane had to be modernised without losing the unique look users had become used to. The challenge was to stick with the same red and grey branding as on the jobsplane.com website, and combine it with the material design guidelines to get a cohesive design identity for the app. We made the logo sleeker, and created a color palette based on the colors used in the logo and website. Website

Username Availability Checker lets you input a username, and almost instantly find out if it’s been taken on Facebook, Twitter, Instagram, Tumblr, Pinterest, Soundcloud and Github. Seriously! Try it out now! No more late nights spent in regret! No more silent tears in the rain! Only one username that you can use anywhere on the internet.

This project was the result of an afternoon spent with my friend Manvendra trying to learn Python. We used Flask, a python framework, to create an API which checks if a username exists on a website. It’s all very clever, and you can read more about it on the project page on Github.
