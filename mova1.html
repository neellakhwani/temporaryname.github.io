Website

Username Availability Checker lets you input a username, and almost instantly find out if it’s been taken on Facebook, Twitter, Instagram, Tumblr, Pinterest, Soundcloud and Github. Seriously! Try it out now! No more late nights spent in regret! No more silent tears in the rain! Only one username that you can use anywhere on the internet.

Visit the website at www.mova1.net.in.

This project was the result of an afternoon spent with my friend Manvendra trying to learn Python. We used Flask, a python framework, to create an API which checks if a username exists on a website. It’s all very clever, and you can read more about it on the project page on Github.
